# @mono/cli

A TypeScript CLI application for the Mono project.

## Installation

```bash
# Install dependencies
npm install

# Build the CLI
npm run build
```

## Usage

```bash
# Run the CLI
npm start

# Or run in development mode
npm run dev

# Run a specific command
npm start -- hello
```

## Available Commands

- `hello`: Displays a greeting message

## Development

```bash
# Run tests
npm test

# Lint code
npm run lint
```
